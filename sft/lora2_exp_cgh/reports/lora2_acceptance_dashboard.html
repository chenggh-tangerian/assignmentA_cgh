Shell（精简后仅这些）

  • common.sh — 路径、环境变量、日志、GPU 显存监控启停
  • 00_init.sh — 准备 data + 生成 Phase1 yaml/manifest/state
  • 01_train_run.sh — 训练一组 LoRA/QLoRA，写 resources
  • 02_predict_eval_run.sh — 预测 + MBPP 评测，写 metrics
  • 03_report.sh — 生成 md/json/html 报告
  • run_all.sh — Phase1→alpha→epoch→报告 一键串起来
  • monitor_gpu.sh — 写 gpu_memory.csv（验收显存指标）

common.sh 函数

  • log_master — 写 master.log
  • run_dir / run_log_dir / ensure_run_log_dir — 目录辅助
  • start_gpu_monitor / stop_gpu_monitor — 显存采集
  • cd_project_root — 切到项目根

setup_data.py

  • main — 拷贝数据并写 dataset_info

  • alpha_for_rank / run_id / build_train_config / build_predict_config / initial_run_state / exp_paths / main — 生成 Phase1 8 组配置

  • create_sweep_run / main — 基于最优配置生成 alpha/epoch sweep

pick_best_run.py

  • pick_best / main — Phase1 选 pass@1 最高

update_state.py

  • recompute_overall / main — 更新 train/predict/eval 状态与指标

collect_train_stats.py

  • parse_* / main — 汇总显存、速度、参数量

  • prune / main — 训完删多余 checkpoint

  • build_report / main — 写 experiment_report.md/.json
