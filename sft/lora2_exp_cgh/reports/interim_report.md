## 1. 核心结论（基于已完成结果）

1. **LoRA/QLoRA 整体有效**：Phase1 8 组中，pass@1 范围 **2.33%–3.50%**，与全量 SFT baseline（**3.11%**）相当或略优。
2. **当前最优**：`r8_tqv_q4_alpha_a1x`（rank=8, alpha=8, QLoRA, q/v）→ **pass@1 = 3.89%**，较 baseline **+0.78 pp**。
3. **Phase1 最优网格配置**：`r8_tqv_q4`（rank=8, alpha=16, QLoRA, q/v）→ **3.50%**。
4. **超参趋势（初步）**：
   - **rank**：rank=8 普遍优于 rank=16（同 target/quant 下）。
   - **target**：q/v 投影 > 全部层（如 rank8 QLoRA：3.50% vs 2.72%）。
   - **量化**：QLoRA 在 q/v 配置下精度与 LoRA 持平或更好，且显存更低（~11 GB vs ~12 GB）。
   - **alpha**：1×rank（alpha=8）目前优于 2×rank（alpha=16）；4×rank 待验证。
5. **效率**：q/v + rank8 组合 adapter 仅 **786K** 可训练参数（~3 MB），训练约 **2h**；rank16+all 可达 **7.6M** 参数、**19 GB** 峰值显存、**4h** 训练。

## 4. 分项分析

### 4.1 精度（MBPP pass@1）

```
Baseline ████░░░░░░░░░░░░░░░░  3.11%
Best     █████░░░░░░░░░░░░░░░  3.89%  (alpha_a1x)
Phase1   ████▌░░░░░░░░░░░░░░░  3.50%  (r8_tqv_q4)
Worst    ███░░░░░░░░░░░░░░░░░  2.33%  (r16_tqv_q4)
```

- 绝对提升幅度小（<1 pp），符合 0.5B 小模型 + MBPP 难度预期。
- **q/v 低秩适配**是精度-效率最佳平衡点；盲目增大 rank 或 target 未带来收益。

### 4.2 显存与速度

| 对比维度 | 更优选择 | 数据支撑 |
|---------|---------|---------|
| 低显存 | QLoRA + q/v + rank8 | 峰值 ~11 GB，adapter 3 MB |
| 高显存 | LoRA + all + rank16 | 峰值 ~19 GB，adapter 29 MB |
| 训练速度 | LoRA q/v rank8 | 1.09 step/s |
| 最慢 | LoRA all rank16 | 0.44 step/s |

QLoRA 以更低显存达到与 LoRA 相近甚至更好的 pass@1（q/v 配置）。

### 4.3 Alpha 缩放（部分结果）

| alpha / rank | pass@1 | 相对 2×rank |
|-------------|--------|------------|
| 1× (8) | **3.89%** | +0.39 pp |
| 2× (16) | 3.50% | baseline |
| 4× (32) | - | 待测 |

**初步判断**：alpha 偏小（1×rank）可能更利于该任务；4×rank 结果出来后需确认是否过拟合或欠拟合。

### 4.4 语法通过率

各组 **93.8%–96.9%**，与 baseline 95.3% 接近。精度差异主要来自**逻辑/测试用例通过**，而非语法生成失败。
